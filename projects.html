Project name: Hospital Appointment Management System

Language(s): MERN Stack


Health Link is a comprehensive hospital appointment management system developed using the MERN (MongoDB, Express.js, React.js, Node.js) stack. This project aims to streamline and enhance the efficiency of the appointment scheduling process within healthcare institutions. The system prioritizes user experience, accessibility, and real-time data management to provide a seamless solution for both hospital and patients. Key functionalities of Health Link include user authentication, appointment creation, modification, and cancellation, as Ill as a comprehensive dashboard for hospital to manage their schedules efficiently. The system employs RESTful APIs for smooth communication betIen the server and the client, ensuring data consistency and reliability. Health Link not only addresses the immediate needs of appointment management but also considers the long-term scalability and adaptability requirements of healthcare institutions. This MERN stack project is designed to enhance the overall healthcare experience by optimizing appointment workflows and fostering better communication between hospital and patients.

Project name: Employee Details Website

Language(s): Angular, Typescript


The aim of this project is work on front-end web development using Angular and Bootstrap web framework, specifically Material Design for Bootstrap and Angular (MDBA). I was required to create a website that displays the employees' details by applying the CRUD (Create, Read, Update and Delete) operations to describe user interface conventions that facilitate viewing, searching, and changing information using computer-based forms and reports. The Employee Details website was created using a specific template created by the DWA Team that is used by all Evonik websites. The first page of the website displays the employees list inside an Angular Datatable. The table contains the employee ID, name and position. The search bar is available to filter the input results accordingly and there are “Create New” button, “Edit” button, and “Delete” button to create, edit, and delete an employee respectively.

Project name: Pharmacy Inventory

Language(s): Java, XML


The project is a mobile application designed to manage a pharmacy's inventory efficiently. Its primary aim is to help the pharmacy store, access, and organize product details while tracking product availability. The application enables users to log in securely, view a list of available products, and check product details. Users can add new products by entering a name, capturing an image, setting a restock date, choosing a supplier, and updating stock status. The app includes a feature to share inventory reports via Bluetooth, Gmail, Messages, or Google Drive. Additionally, it offers a show/hide subtitle function to display the total product count and operates in both portrait and landscape modes. A small pill icon appears beside restocked items for easy identification. The system was tested for functionality, security, and usability, confirming that product listing, login authentication, and interactive buttons worked as intended. This user-friendly tool improves pharmacy inventory management through accessible and organized mobile technology.

Project name: Vehicle Price Predictor

Language(s): Python, Flask


Vehicle Price Prediction is a machine learning web application that estimates the resale price of used vehicles based on key details like make, model, year, mileage, fuel type, transmission, and engine size. The project uses a Random Forest Regressor trained on a dataset of popular car models in Malaysia. Built with Python and Flask, it features a clean, user-friendly interface with dropdowns for categorical inputs and instant price predictions. This project demonstrates practical AI application in the automotive sector, covering data preparation, model training, evaluation, and deployment within an interactive web-based environment

Project name: Vehicle Dynamics Simulation

Language(s): MATLAB, Simulink


This Vehicle Dynamics Simulation models both longitudinal and lateral vehicle behavior using MATLAB and Simulink. It simulates how a vehicle responds to traction, aerodynamic drag, rolling resistance, and cornering forces during acceleration, braking, and steering. The longitudinal dynamics calculate speed over time based on a simplified force balance equation. The lateral dynamics use a bicycle model to estimate yaw rate and sideslip angle in response to a constant steering input. The project includes modular MATLAB scripts for simulations, parameter files for vehicle constants, and plotting scripts to visualize speed, yaw rate, and sideslip angle over time. A complementary Simulink model visually simulates these dynamics using interactive block diagrams and real-time scope displays.
